# Design Algorithms (DA) - Practical Exercises

This repository contains solutions to 6 algorithmic problems from my Design Algorithms course by Dr. Abam at Sharif University of Technology (SUT) in Semester 5 (Fall 2025).

![Language](https://img.shields.io/badge/language-C++%20%7C%20Python%20%7C%20Java-blue)


## 📁 Problems

| # | Topic | Description |
|---|-------|-------------|
| 1 | Greedy | Lipschitz Sequence Maximum Weighted Sum |
| 2 | Divide & Conquer | 3D Dominance Counting |
| 3 | DP & Backtracking | DNA Sequence Design (Aho-Corasick + DP) |
| 4 | DFS | Non-Descending Sequences (Brute Force) |
| 5 | Bipartite Matching | Minimum Operations to Clean Grid |
| 6 | String Compression | Minimum Compressed String Length |

- **Note**: Each folder contains the problem description, solution code, and an explanation of the implemented algorithm.

## 🛠️ Languages

- C++ (primary)
- Java (some problems)
- Python (some problems)
